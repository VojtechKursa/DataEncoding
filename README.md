# DataEncoding Library
A .NET Standard 2.0 library for encoding/serializing and decoding/deserializing data using formats: DER, PEM, JSON and XML.
(Even though the implementation of the last one is quite horrible)

## Building the library
You can build the library by opening either the *.csproj* or *.sln* file using Visual Studio or any equivalent program and building the project.
Alternatively you can download a build from the [releases section of the GitHub repository](https://github.com/VojtechKursa/DataEncoding/releases).
